Lithium Roadmap

Last Lithium Feature Update: 1/18/2022

These features are still being updated and improved, but have the most significant functionality in the engine.

  • Filesystem Manager - Abstracts filesystem operations and handles loading some file formats.
  • Dynamic Shader Loader - Links related vertex and fragment shaders together and binds them to input from material files.
  • Event-Driven GUI System - Interface set up in a way that mimics Java AWT.
  • C++ Scripting System - Developers may write C++ scripts to define the internal behavior of their application.
  • HDR and Post-Processing Support - Supports High Dynamic Range (HDR) rendering and post-processing effects such as Gaussian Blurs, Bloom and FXAA.
  • Material System - Handles loading material files which define a set of parameters to tell the engine how to draw an object. Supports albedo, specular, normal, ambient oucclusion and reflection mapping.
  • Shadow Mapping - Renders shadow maps which are used in shaders to create shadows in a scene.
  • Terrain Rendering - Rendered using a heightmap for vertex positions and splatmap for texture blending.
  • Particle Systems - Uses instanced rendering to be able to draw a large number of objects with little performance impact.
  • Authoring Tools - A set of tools for developers to create assets for their game outside of code.

These features are currently being implemented and do not have full functionality.

  • Map System - Files to be loaded by the engine which define both static and dynamic world geometry outside of C++ code. Maps are compiled from a source file into a binary format containing metadata, entity data, static world geometry, and custom resources.
  • Animation System - A system to automatically handle transformations of objects within a scene.
  • SSAO - Screen-Space Ambient Occlusion post-processing effect

These features I want to eventually add sometime in the future in no particular order.

  • Texture Animations
  • PBR Rendering
  • Map Editor
  • Planar Reflections
  • Screen-Space Reflections
  • Depth-of-Field
  • LScript Scripting
  • Custom Libraries
